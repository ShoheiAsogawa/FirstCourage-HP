# FirstCourage Brand Buying Site

株式会社ＦｉｒｓｔＣｏｕｒａｇｅの信用形成・公式LINE登録獲得を目的にしたNext.jsサイトです。

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- lucide-react

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content Management

公開前に差し替える会社情報、電話番号、公式LINE URL、実績、口コミ、参考価格、SEO記事、画像生成プロンプトは `src/data/site.ts` に集約しています。

戦略・ワイヤーフレーム・CTA設計・画像プロンプト・Vercel手順は `docs/strategy-and-implementation.md` を参照してください。
